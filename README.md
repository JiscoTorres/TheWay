# TheWay

## Alunos: João Francisco Torres e Marcus Vinícius R. Bacelar.

Sistema de Gestão Comercial desenvolvido para auxiliar na administração de uma **loja de suplementos alimentares**.

O sistema permite o gerenciamento de **clientes, produtos, controle de estoque e registro de vendas**, centralizando todas as informações da loja em uma única plataforma.

Além disso, o sistema possui um **módulo de recomendação de suplementos**, que auxilia clientes e funcionários na escolha de produtos com base em objetivos como **hipertrofia, emagrecimento ou melhora de desempenho físico**.

## ⚙️ Funcionalidades

- Autenticação de usuários com **JWT**
- Cadastro e gerenciamento de **clientes**
- Cadastro e gerenciamento de **produtos**
- **Controle de estoque**
- **Registro de vendas**
- **Relatórios de vendas por período e por cliente**
- **Recomendação de suplementos** com base em objetivos

## 🏗 Arquitetura

O sistema foi desenvolvido utilizando **arquitetura em camadas**, separando as responsabilidades entre:

- **Camada de Apresentação** – Interface web acessada pelo usuário
- **Camada de Aplicação (API)** – Responsável pelos endpoints da API REST
- **Camada de Lógica de Negócio** – Regras do sistema (vendas, estoque, recomendações)
- **Camada de Persistência de Dados** – Comunicação com o banco de dados

Essa abordagem facilita a **organização do código, manutenção e escalabilidade do sistema**.
